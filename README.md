# COWY - Frontend with React

This is a blogging website for portfolio purpose. It is built using React + Strapi + MongoDB.

This repo is the source code for the fronted only

Technologies involved:

- react
- react-redux
- react-router-dom
- react-helmet
- redux-persist
- styled-components
- axios
- js-cookie
- slugify
- draft-js
- react-draft-wysiwyg
